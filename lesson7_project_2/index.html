Overview

Have you ever wondered how exactly a web page is created? There is a lot of coding that is done to make a website work. What you see when you open a web page is done by the work of front-end developers.

Front-end development is the process of using HTML and CSS for a website so that a client is able to interact with and use it. Front-end development is what is responsible for everything we see and interect while on a web page. To do this, front-end developers mainly use HTML, CSS, and JavaScript to develop and design these web pages. It is also important that a front-end developer focuses on making sure that their site is cross-browser compatible.

There are many skills that are essential to becoming a successful front-end developer. Some of these include knowing how to use certain buliding blocks of web coding such as CSS, HTML, and JavaScript. Knowing the testing and debugging processes is also very important to becoming a front-end developer. Other skills that are important to have is knowing how to make your site responsive and compatible across all browsers and devices.

Jobs within this particular area of study are very rewarding. The average front-end developer gets paid about $78k a year. They can also earn a salary of over $100k after years of experience!

Important Languages/Programs to Become Familiar With:

HTML HTML is a Hyper Text Markup Language. It is used to define the structure and layour of a webpage.
CSS CSS stands for Cascading Style Sheets. CSS is used to describe how HTML elements are to be displayed.
JavaScript JavaScript is a programming language that can be used to update, change and manipulate HTML and CSS.
jQuery jQuery is a JavaScript library used to simplify JavaScript programming
