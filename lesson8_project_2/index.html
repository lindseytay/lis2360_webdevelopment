HTML. Structure and layout.

HTML is a computer language that allows people to create webpages. HTML consists of tags that allow people to add structure and layout to their code and create websites that can be seen by other people.

CSS. Design and color.

CSS allows people to take their HTML document and design it even further by adding colors, fonts, and other styles to personalize their webpage.

JS. Change and manipulate.

To go even further, JavaScript can be used manipulate the HTML and CSS contents by adding animations and interactive buttons and etc.
